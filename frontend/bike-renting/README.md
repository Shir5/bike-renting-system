# Bike Renting Expo client

Инструкции по установке, переменным окружения, запуску, сборке и тестированию находятся в [корневом README](../../README.md).

Минимальный локальный запуск:

```bash
cp .env.example .env
npm ci
npm start
```

Базовый URL backend задаётся переменной `EXPO_PUBLIC_API_URL`.
